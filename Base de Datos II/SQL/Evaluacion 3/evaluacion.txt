Instrucciones: Responda las siguientes preguntas utilizando sentencias SQL con INNER JOIN.

3. Obtenga una lista de los productos vendidos en cada venta, mostrando el ID de la venta, el nombre del producto y la cantidad vendida. (Ventas - DetalleVentas - Productos)

5. Muestre el nombre de los productos comprados por el cliente 'Maria Gomez'. (Clientes - Ventas - DetalleVentas - Productos)

6. Liste todas las categorías de productos que se han vendido alguna vez. (Categorias - Productos - DetalleVentas)

7. Muestre el detalle de la venta con ID 1, incluyendo el nombre del cliente, el nombre del producto, la cantidad y el precio unitario. (Clientes - Ventas - DetalleVentas - Productos)
