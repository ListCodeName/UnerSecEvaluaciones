Evaluación 1 de Nuevas Tecnologías

A continuación se pedirá que creen una web para un entorno educativo, que permita llevar los temas dados por los profesores en una determinada materia. Se deberá poder crear una materia, asignar un profesor o varios profesores a la misma, y luego poder guardar día a día los temas, pudiendo repetir los temas mediante un botón de ayuda. Deben poder modificarse los campos en caso de cometer algún error de carga. En caso de haber feriados estos se mostrarán en la lista de otro color para identificarlos mejor visualmente.

  • Crear un diseño web en chatgpt (html y css) que permita maquetar el formato, estilos y los elementos necesarios para la web
  • Llevar el todo el código generado y utilizar antigravity para darle las funcionalidades necesarias.
  • Realizar la base de datos mediante un archivo JSON, y las interacciones con la página mediante Javascript.
  • Realizar las pruebas y corrección de errores de ser necesario para mostrar una implementación completa y funcional
